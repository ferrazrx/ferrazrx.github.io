Parot Party

Design

The ParrotParty website design was created to reflect the variety and vibrant colours that convey ‘bright parrot colouration”. The idea was to use the same colours (Green, Orange, Blue, Yellow) of the birds feather to create the warm feeling that those animals bring.

See Document

The home page was divided into 7 sections:

  • home
  • navigation
  • photo contest
  • benefits
  • blog Posts
  • testimonials
  • team Members and
  • footer
Each section was divided by different colours to help the user to differentiate the different subjects. The main idea of this page is to present images and content in a smooth way. All pages use a centralized alignment for the elements and texts.

The resources page was divided into 5 sections: Search Navigation, photocontest, subscription Ad, List of content and footer. The main idea of this page is to offer the user the possibility to search content in the website with more options than just the usual search bar. The user can scroll and find the content listed as well.

All the pages were created with the concept of “mobile fist” helping the mobileusers to have a simple and efficient navigation.
